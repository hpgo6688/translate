# Immersive Translate 类浏览器插件 - 功能拆分文档

## 一、核心翻译功能

### 1.1 网页双语对照翻译
- **段落级双语显示**：在原文段落下方/旁边插入译文，保留原文不替换
- **多种显示样式**：
  - 上下对照（译文在原文下方）
  - 左右对照（原文左、译文右）
  - 仅显示译文（隐藏原文）
  - 鼠标悬停显示原文
- **样式自定义**：译文颜色、字号、下划线、虚线框、模糊效果等多种视觉标记

### 1.2 智能内容识别
- 自动识别正文区域，跳过导航栏、页脚、广告
- 识别代码块并跳过翻译
- 保留原文链接、图片、格式标签
- 智能合并被拆分的句子（如跨标签的句子）
- 数学公式、专业术语保护

### 1.3 划词翻译
- 选中文本后弹出翻译气泡
- 输入框翻译（支持长文本粘贴翻译）
- 快捷键触发翻译

### 1.4 鼠标悬停翻译
- 鼠标悬停在段落上显示翻译
- 配合修饰键（Ctrl/Alt/Shift）触发，避免误触

## 二、翻译服务支持

### 2.1 内置免费翻译引擎
- Google 翻译
- 微软 Bing 翻译
- DeepL Free
- 腾讯交互翻译、有道、百度等

### 2.2 AI 大模型翻译
- OpenAI（GPT-4/GPT-3.5）
- Claude（Anthropic）
- Gemini（Google）
- DeepSeek、通义千问、文心一言、Kimi 等国产模型
- 自定义 API 端点（兼容 OpenAI 格式的第三方服务）
- 本地模型（Ollama 等）

### 2.3 翻译服务管理
- 多服务并行配置，按场景切换
- API Key 管理与加密存储
- 请求频率/并发数控制
- 失败自动降级到备用服务
- 翻译额度/Token 统计

## 三、文档与文件翻译

### 3.1 PDF 翻译
- 上传 PDF 双语对照阅读
- 保留排版（图、表、公式位置）
- 导出双语 PDF

### 3.2 EPUB 电子书翻译
- 上传 EPUB 双语对照阅读
- 导出双语 EPUB
- 章节进度保存

### 3.3 字幕翻译
- YouTube/Netflix/Bilibili 等视频网站双语字幕
- 字幕文件（SRT/VTT/ASS）上传翻译
- 实时字幕对照

### 3.4 其他文档
- Word/TXT/Markdown 文件翻译
- 网页快照导出双语 PDF/HTML

## 四、特定场景增强

### 4.1 输入框增强翻译
- 在任意网站输入框中输入中文，按快捷键自动翻译为英文（或反向）
- 适用于 Twitter、Gmail、Discord 等

### 4.2 视频字幕实时翻译
- 支持主流视频平台原生字幕双语化
- AI 实时转录并翻译无字幕视频

### 4.3 图片 OCR 翻译
- 网页图片中的文字识别并翻译
- 截图翻译

## 五、用户体验功能

### 5.1 语言配置
- 源语言自动检测
- 目标语言自定义（支持几十种语言）
- 多目标语言（同时翻译为多种语言）

### 5.2 触发方式
- 一键翻译按钮（页面浮动按钮）
- 快捷键自定义
- 自动翻译规则（按域名/语言）
- 滚动到可视区域时按需翻译（性能优化）

### 5.3 网站规则配置
- 域名白名单/黑名单
- 针对特定网站的 CSS 选择器自定义（精准定位翻译区域）
- 永不翻译的元素列表

### 5.4 词汇与术语管理
- 个人术语表（特定词强制译为指定结果）
- 不翻译词列表（保留原文）
- AI Prompt 自定义（控制翻译风格、语气）

## 建议的开发优先级（MVP → 完整版）

**P0（MVP 核心）**：网页双语翻译、Google/DeepL 免费引擎、一键翻译按钮、基本设置面板、翻译缓存

**P1（基础增强）**：AI 模型接入（OpenAI/Claude）、划词翻译、输入框翻译、网站黑白名单、自定义 Prompt

**P2（差异化竞争）**：PDF 翻译、视频字幕双语化、EPUB 翻译、术语表

**P3（高级功能）**：图片 OCR 翻译、多模型对比、学习辅助、云同步、付费体系
